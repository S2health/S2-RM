= Ordered Package: Temporal

== Overview

The temporal types from the `data_types.quantity` package provide data types for date, date/time, time and duration, based on {iso_8601_official}[ISO 8601:2019 standard^]. They are formally defined in terms of the primitive `Date`, `Date_time`, `Time` and `Duration` types from the BASE component.
